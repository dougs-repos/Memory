# Memory

- [US Paperback Vol 2 - design tools for modern mnemonics](https://www.amazon.com/dp/B0C2SM3KG7)
- [US Paperback Vol 3 - 7711 Bible Memorization by Chapter Themes](https://www.amazon.com/dp/B0C8QBJZ8D)
